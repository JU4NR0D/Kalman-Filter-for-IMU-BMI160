# Kalman-Filter-for-IMU-BMI160
Implementación de un filtro de Kalman lineal para la IMU BMI160 con ESP32, que fusiona datos de giróscopo y acelerómetro para estimar ángulos de alabeo y cabeceo en tiempo real. Incluye adquisición I2C, caracterización del sensor y obtención de matrices del filtro desde MATLAB.
